## Node. js 是什么
---
**Node.js** 是一个基于Chrome V8 JavaScript引擎的JavaScript运行时环境，让JavaScript可以在服务器端运行。

假如没有 Node. js，JavaScript 只能在浏览器上运行，而 Node. js 可以让 JavaScript 像 Python、Java 一样，能够直接在电脑上运行
